Constructor Detail

customProperties

customProperties(<string> deliveredToProperty,<string> statusProperty,<string> dsnSummaryProperty,<string> flagsProperty,<string> mdnDisplayedSummaryProperty,<string> mdnDeletedSummaryProperty)
    format for deliveredToProperty (x-nviewer-to) property: 

    	field separator: ";"
	row separator: "\n\t"

	fields:                   type
	final recipient           string
	flags                     number
	DSN Count                 number
	DSN                       string
	DSN message-Id            string
	MDN Count                 number
	MDN                       string
	MDN message-Id            string

	final recipient;flags;DSN Count[;DSN;DSN message-Id]*;MDN Count[;MDN;MDN message-Id]*
    example: 
    	daniel@vraimentbidon.org;0;1;delivered;<123456@vraimentbidon.org>;1;displayed;<AZERG@vraimentbidon.org>\n\t
	sersim@vraimentbidon.org;1;1;delivered;<543211@vraimentbidon.org>;2;displayed;<GBI4E@vraimentbidon.org>;deleted;<AZT564@vraimentbidon.org>

    format for flagsProperty (x-nviewer-flags) property:

    NOT FROZEN 

    	b ...x  time out (global)
	b xxx.  customProperties version
    Parameters:
      deliveredToProperty - x-nviewer-to property
      statusProperty - x-nviewer-status property ("good","middle", "bad")
      dsnSummaryProperty - x-nviewer-dsn-summary property (number/total)
      flagsProperty - x-nviewer-flags property
      mdnDisplayedSummaryProperty - x-nviewer-mdn-displayed-summary property (number/total)
      mdnDeletedSummaryProperty - x-nviewer-mdn-deleted-summary property (number/total)
    Version: 0.9.4
