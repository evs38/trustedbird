getOriginalMsgId

string getOriginalMsgId()
    Get original message-id for this DSN.
    Returns:
      return message-id fields from the original message

      In order to conform to RFC 822, the Message-ID must have the format 

      "<" "unique" "@" "full domain name" ">"

      Example: 

      <20080213131118.4DD991FF88@mydomain.org>

getReportFromFields

deliveryReport getReportFromFields(<string> dsnFieds)
    Return one delivery Report.
    Parameters:
      dsnFieds - group of per-recipient fields.

      The syntax for the group of per-recipient fields is as follows (RFC 3464):

      • [ original-recipient-field CRLF ]
      • final-recipient-field CRLF
      • action-field CRLF
      • status-field CRLF
      example: 
       	Original-Recipient: rfc822;louisl@larry.slip.umd.edu
       	Final-Recipient: rfc822;louisl@larry.slip.umd.edu
       	Action: failed
       	Status: 4.0.0
       	Diagnostic-Code: smtp; 426 connection timed out
    Returns:
      A deliveryReport object or null if is not a DSN
