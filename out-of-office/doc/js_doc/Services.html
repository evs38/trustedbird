emailCheck

Object emailCheck( address, log )
    Check the address mail validity in regards of the RFC 2822 If this address is valid the function return the short address
    Parameters:
      string - ) Address mail to check.
    Returns:
      (string) null for an invalid address else return the short address like user@domain.ext
    Author: Olivier Brun / BT France 1.1.Services.prototype.3 = Amended;error messages and allowed script to deal with new TLDs 1.1.Services.prototype.2 = Fixed;a bug where trailing . in e-mail address was passing (the bug is actually in the weak regexp engine of the browser; I simplified the regexps to make it work). 1.1.Services.prototype.1 = Removed;restriction that countries must be preceded by a domain, so abc@host.uk is now legal. 1.Services.prototype.1 = Rewrote;most of the function to conform more closely to RFC 822. 1.Services.prototype.0 = Original;

localizeString

Object localizeString( localeFileName, message, arrayValue )
    Localize stringBundle. This function can be insert dynamically parameters in the string. The user that define the localized string insert tag as %1, %2 ... The function parse the string and insert each value from the third parameter arrayValue. If the value is not define the value take the value undefined. If the array is too small the value stay the tag %7 for example. If the array is too big nothing append.
    Parameters:
      string - ) localeFileName String bundle label id.
      string - ) message String id define in the stringBundle file
      array - ) arrayValue Array of the value to replace in string. Value in string are defined with tag %x where x is the value index.
    Returns:
      (string) Localized string message to use by the caller.
    Author: Olivier Brun / BT France
