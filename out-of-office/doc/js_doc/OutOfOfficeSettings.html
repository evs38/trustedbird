latestScript

string latestScript
    Latest script core.

notificationEnable

boolean notificationEnable
    The notificationEnable field activate the automatic response in case of the 'Out Of Office' of the user. For each mail message received in the mailbox of the user account, all sender will recieve an out of office notification.

notificationMessage

string notificationMessage
    Definition of the message sent for each sender of a mail message to this mailbox account.

notificationSubject

string notificationSubject
    Value define to put the default subjet for the notification

prefURI

string prefURI
    Preference URI of the extension.

redirectionDestinationAddress

string redirectionDestinationAddress
    The destinationAddress field indicates the address mail to redirect the received mail for the selected account.

redirectionEnable

boolean redirectionEnable
    Redirection activation field. It allow the user to redirect his mail to another destination address define in the destinationAddress field.

redirectionKeepMessage

boolean redirectionKeepMessage
    Keep message allow the user to keep the original message to his own mail box. If this field is not activated the redirected mail message will be removed from the user mailbox.

redirectionShortMailAddress

string redirectionShortMailAddress
    The short destinationAddress to redirect the received mail for the selected account. This address looks like the following format 'user

scriptCore

string scriptCore
    Object that contains the generated data to build the runtime script. This field will be sent to the sieve server in the file named 'scriptOutOfOffice'.
